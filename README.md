# download-images

## Install:
> npm install

## Run:
> execute bin.js

Will execute multi-thread version (mainmt.js)

## Compile:
> npm install -g pkg

### For Windows
> ./make.exe.bat

### For Linux
> ./make.sh.bat
> chmod +x baixar_lista


output will be on root folder as ./baixar_lista.exe (win) or ./baixar_lista (unix)
